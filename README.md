# thesis-appendix-materials

Questo repository raccoglie materiale supplementare ed esplorativo a supporto della tesi.

Sono inclusi frammenti di codice, piccoli prototipi, appunti ed esperimenti ausiliari utilizzati durante lo sviluppo del lavoro e richiamati nelle appendici.  
Il contenuto ha finalità prevalentemente didattiche e illustrative e riflette passi intermedi e implementazioni di supporto alla trattazione teorica.

Il materiale **non costituisce un progetto software strutturato**, né è pensato per un utilizzo produttivo o per valutazioni prestazionali.  
L’obiettivo è fornire trasparenza, riproducibilità e un contesto operativo aggiuntivo rispetto alla discussione teorica.

## Struttura

Il repository è organizzato in sottocartelle tematiche, ciascuna riferita a uno specifico argomento o a una sezione delle appendici della tesi (ad esempio strutture di indicizzazione, trasformazioni o dimostrazioni algoritmiche).

## Note

- Il codice privilegia chiarezza e aderenza alla teoria rispetto all’efficienza.
- Alcuni contenuti possono essere sperimentali o volutamente semplificati.
- La struttura del repository può evolvere durante la finalizzazione della tesi.
