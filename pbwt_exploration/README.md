# PBWT — esplorazione pratica

Questa cartella contiene appunti ed esperimenti svolti durante il tirocinio
per esplorare l’utilizzo pratico dell’implementazione originale della
Positional Burrows–Wheeler Transform (pbwt) di Richard Durbin.

Il materiale ha carattere puramente esplorativo e operativo
(installazione, conversione di file VCF, esempi di utilizzo e output),
ed è stato utile per maturare una comprensione pratica dello strumento.

Non costituisce materiale direttamente utilizzato o discusso nella tesi.
