# FM-index — Dimostrazione didattica

Questa cartella contiene un’implementazione didattica della Burrows–Wheeler Transform
e del meccanismo di backward search dell’FM-index, sviluppata a supporto del Capitolo 3 della tesi.

Il codice è pensato per rendere operativi i concetti teorici discussi nel testo
(suffix array, BWT, array C, operazione Occ, LF-mapping e ricerca backward),
privilegiando chiarezza e tracciabilità rispetto all’efficienza.

L’output di esempio riportato in `output_example.txt` è ottenuto eseguendo il programma
sulla stringa di input `banana`.
